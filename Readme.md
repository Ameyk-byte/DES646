# Nuero Personal Assistant (Personalized Learning Recommender)

## Overview
# Neuro – AI Voice Assistant with Personalized Learning Recommender

Neuro is an intelligent personal voice assistant built by **Team Synapse**
using Python, speech recognition, Cohere LLM-routing, browser UI (EEL),
automation functions, and machine-learning based learning recommendation.

✅ Speech → NLP → AI Decision Model  
✅ Personalized Learning Recommender  
✅ Real-time search, automation, IoT control  
✅ Audio-reactive neon 3D UI

---

## ✅ Features

| Capability | Description |
|------------|-------------|
| ✅ Wake Word Detection | Say **"Neuro ..."** to activate |
| ✅ Personalized Learning Recommender | Recommends courses, tutorials, videos + creates weekly study plan |
| ✅ Chatbot for general questions | Uses Cohere LLM |
| ✅ Real-time search engine | Uses Google Search API + web-scraping |
| ✅ App Automation | Open / close apps & websites |
| ✅ IoT Control (Simulated) | "Turn on light", "Set AC to 22°C", etc. |
| ✅ AI Image Generation | “Neuro generate image of a tiger” |
| ✅ Voice Output | Responds with speech using edge-tts |
| ✅ Browser-based Visual UI | Glowing audio-reactive sphere + learning panel |

---

## ✅ Tech Stack

- **Python 3.10**
- Cohere LLM (query classification + chatbot)
- EEL (UI + Python ⇆ Web communication)
- SpeechRecognition + PyAudio (voice to text)
- edge-tts (text to speech)
- scikit-learn + cosine similarity (recommender)
- Three.js & Canvas (visual sphere)
